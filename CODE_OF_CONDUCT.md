## Unacceptable Behavior

The following are not tolerated:
- Hate speech, harassment, or discrimination of any kind.  
- Spam, self-promotion, or solicitation.  
- Attempts to gain unauthorized access, reverse-engineer, or redistribute the project code.  
- Misrepresentation of affiliation with the project or its maintainers.

## Scope

This Code of Conduct applies to all interactions in:
- GitHub issues, pull requests, and discussions.  
- Any official or public communications related to the project.

## Enforcement

All moderation decisions are made by the repository owner and designated collaborators.  
Actions that violate this Code may result in:
1. Comment removal or issue locking.  
2. Temporary or permanent ban from repository interaction.  
3. Reporting to GitHub if necessary.  

Enforcement is at the sole discretion of the repository owner.
